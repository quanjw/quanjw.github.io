Sep 27, 2019

一、GitHub创建个人仓库

按照链接 Github Page 创建仓库
仓库的名字设定为：自己注册时的用户名 +.github.io

我的用户名为：quanjw
那么我的仓库名字就是： quanjw.github.io

地址:git@github.com:quanjw/quanjw.github.io.git

二、安装Git

https://www.git-scm.com/download/

下载安装完成后，在命令行里输入 git –version 看看版本并测试是否安装成功。

三、将本地的 Git 与 GitHub 帐号关联

git config –global user.name “GitHub用户名”
git config –global user.email “GitHub注册邮箱”

生成ssh密钥文件：

复制里面的内容，打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：
Title为标题，任意填即可，将刚刚复制的id_rsa.pub内容粘贴进去，最后点击Add SSH key。

测试是否成功添加公钥

成功会提示如下：

Hi *! You’ve successfully authenticated, but Git
Hub does not provide shell access
