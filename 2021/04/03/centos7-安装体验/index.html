centos7 安装体验

Posted on 2021-04-03

前言

新配置的PVE，装了128G的ssd，终于有机会可以放更多虚拟机了，所以这次安装了centos7作为家里的服务器，以供体验和学习。

问题：

网络配置

首先是网络配置的问题，网卡的配置在/etc/sysconfig/network-scripts/路径下，ifconfig 查看，得知网卡是eth0，vi /etc/sysconfig/network-scripts/ifcfg-eth0编辑。

BOOTPROTO=static
IPADDR=192.168.51.3
GATEWAY=192.168.51.1
DNS1=8.8.8.8
NETMASK=255.255.255.0
DNS1=114.114.114.114
ONBOOT=yes

#主要是这几项

然后重启网络 systemctl restart network ping 一下百度一般就可以了。 ping www.baidu.com

更换CentOS-Base.repo

想安装一个nginx，官方的好像没有，搜索了下，换成阿里的

下载 wget (吐槽下，真的要啥啥没有)，yum install wget
备份 mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak20210403
下载新的源wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
安装epel源 yum install epel-release
此时 可安装nginx了 yum install nginx
防火墙

后面路由器做了端口转发，想从外网访问内网centos7，发现不通。

首先是看了下nginx服务启动了没有。emmm 没有。启动服务systemctl start nginx ；设置开机启动systemctl enable nginx
仍然不通。然后在本机curl ，是有信息的。查看了防火墙 firewall-cmd –state 是running的，查看开放的端口 firewall-cmd –zone=public –list-ports 80没有开放。开放端口firewall-cmd –zone=public –add-port=80/tcp –permanent ,重启防火墙systemctl restart firewalld.service
此时顺利访问了
